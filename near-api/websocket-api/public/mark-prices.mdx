---
title: "Mark prices"
---

- Push interval: 1s

```json
{
    "id": "clientID",
    "topic": "markprices",
    "event": "subscribe"
}
```

**Parameters**

| Name   | Type   | Required | Description               |
| ------ | ------ | -------- |---------------------------|
| id     | string | Y        | id generate by client     |
| event  | string | Y        | `subscribe`/`unsubscribe` |
| topic  | string | N        | `markprices`              |

<RequestExample>

```json Response
{
   "id":"clientID9",
   "event":"subscribe",
   "success":true,
   "ts":1682242805633
}
```
</RequestExample>
