---
title: "Get faucet USDC(Testnet only)"
---

Limit: 5 requests per Account

<RequestExample>

```bash
curl -X POST 'https://testnet-operator-evm.orderly.org/v1/faucet/usdc' \
    -H 'Content-Type: application/json' \
    -d '{"chain_id": "80001", "user_address": "0x036Cb579025d3535a0ADcD929D05481a3189714b"}'
```
</RequestExample>

`POST https://testnet-operator-evm.orderly.org/v1/faucet/usdc`

Receive 1,000 USDC in the Testnet environment. Each account may only use the faucet a maximum of 5 times.

<ResponseExample>
```json
{
    "success": true,
    "timestamp": 1688606744795
}
```
</ResponseExample>
Parameters

| Name         | Type   | Required | Description                                            |
|--------------|--------|----------|--------------------------------------------------------|
| chain_id     | string | Y        | The chain ID that the test USDC should be deposited to |
| user_address | string | Y        | The address of the user account                        |
