---
title: "Get Orderly Key"
---

Limit: 10 requests per 1 second per IP address

<RequestExample>
```bash
curl 'https://api-evm.orderly.org/v1/get_orderly_key?accountId=0xb129e074a17dfba8d652ae1eca21c7d6bb6904f1aa693f4886d50db170933a46&orderlyKey=ed25519:7tEoJo5hMBKBrQsqmc8yw1xNfoQCFQBVwQT1eFafRNRf'
```
</RequestExample>
`GET /v1/get_orderly_key`

Check the validity of an Orderly access key attached to the account.

<ResponseExample>
```json
{
    "success": true,
    "data": {
        "orderly_key": "ed25519:7tEoJo5hMBKBrQsqmc8yw1xNfoQCFQBVwQT1eFafRNRf",
        "scope": "read,trading",
        "expiration": 1689086651947,
        "tag": "some user defined tags" // Optional, could be null if there wasn't a tag
    },
    "timestamp": 1689146723102
}
```
</ResponseExample>
**Parameters**

| Name       | Type   | Required | Description                              |
|------------|--------|----------|------------------------------------------|
| accountId  | string | Y        | The account id of the user               |
| orderlyKey | string | Y        | The public key of the Orderly access key |
