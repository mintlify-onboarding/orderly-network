---
title: "Get TradingView history bars"
---

<RequestExample>
```bash
curl 'https://api-evm.orderly.org/v1/tv/history?symbol=PERP_NEAR_USDC&resolution=D&from=0&to=1683881314470'
Limit: 10 requests per 1 second per IP address
```
</RequestExample>
